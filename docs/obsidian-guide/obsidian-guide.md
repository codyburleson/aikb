# Obsidian Guide for AIKB-Conformant Knowledge Vaults

## Obsidian Plugins

We recommend the following plugins for AIKB-conformant knowledge vaults. The AIKB Reference Vault already has these plugins installed, but you can install them in any vault.

- In Obsidian, go to Settings > Community plugins: Turn on community plugins
- Restricted mode: Turn on and reload
- Community plugins: Browse
- Search the following plugins, install and enable each:
  - Tag Wrangler, by [PJ Eby](https://github.com/pjeby)
